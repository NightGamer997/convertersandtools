ℹ️ Was ist Binär?

Binärcode ist ein Zahlensystem, das nur zwei Ziffern verwendet: 0 und 1. Computer verwenden Binärcode, um alle Arten von Daten zu speichern und zu verarbeiten.

Jedes Zeichen wird als eine Sequenz von 8 Bits (ein Byte) dargestellt. Zum Beispiel: Der Buchstabe "A" ist 01000001 in Binär.
